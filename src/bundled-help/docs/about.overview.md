**Document Viewer** is a browser app for reading and editing **doc-tree** documentation projects.

Each page is a JSON file (`.p`) made of **components** — blocks of text, images, Markdown, or interactive before/after animations. Components on different pages can be **linked** so clicking one highlights related content elsewhere.

You can work **offline** with a local folder, **collaborate** via Supabase remote storage, or start a **new draft** and export later.

**Live demo:** [giapsoft.github.io/document-viewer](https://giapsoft.github.io/document-viewer/)
